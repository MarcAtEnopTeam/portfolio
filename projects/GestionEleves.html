Projet Afip web Gestion des Elèves

Gestion des Elèves

Dans le cadre de ma formation, suite à 4 semaines de formation sur Php5, MySql, programmation orienté objet, Pattern Mvc, l’exercice d’évaluation proposé a été la production d’un site de gestion des élèves.

Le déroulement de l’exercice était de quatre heures pour la production de l’interface graphique, les diagramme mcd et mld pour la base de données et le reste de la semaine pour le finaliser. Je devais recréer un site de gestion des élèves et l’interface de gestion demandée, le site étant découpé en 3 pages de gestion : ajout, suppression, modification.

Actions effectuées pour l’interface utilisateur :

  • Formulaire d'insertion d'un élève à une formation.
  • Création de possibilité de choix du professeur en fonction de la formation choisie.
  • Création d'une page dynamique de modification des stagiaires, de leur formation, gestion du planning des professeurs, et date de formation.
  • Création d'une page de suppression de l'étudiant ou d'un cours enregistré.

Technologies utilisées :

  • Html
  • Bootstrap
  • JavaScript
  • Php
  • MySql

Pattern:

  • Mvc

IDE:

  • Netbeans

Serveur:

  • Xampp

Deadline do projet : une semaine(Lundi-Vendredi).


Lors d’avoir reçu le sujet du projet, j’étais censé à développer la base de données et la valider avec le professeur.

Une fois cette étape conclue, j’ai commencé le développement direct avec NetBeans.

J’ai commencé par la création d'Insertion, de suite la modification et finalement la suppression.

Finie la partie html, Php, j’ai grisé des champs selon le cours choisit avec Javascript.

Pour finaliser, j'ai rajouté les contrôles de données saisies afficher les erreurs à l'utilisateur.


Mon point de Vue

Ce projet a eu une énorme importance pour moi car c'était mon premier projet web, orienté objet et avec une base de données. Plusieurs difficultés on était rencontré tout au long du projet, notamment l'implémentation de la programmation orienté objet, le développement avec JavaScript et quelques problèmes avec la base de données MySql.

Avec beaucoup de motivation et effort, j'ai reussi à faire face aux problemes rencontrés et certains avec l'aide de quelques sites sur l'internet.

Pour conclure, ce projet à servi à comprendre mieux le mode de programmer pour le web et à acquérir de l'expérience sur la programmation orienté objet
