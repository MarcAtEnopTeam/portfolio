Projet Afip La Banque

La Banque

Dans le cadre de ma formation, suite à 4 semaines de formation sur Java chez afip, l’exercice d’évaluation proposé a été la production d’une application bancaire.

Le projet consistait en:

  • Création d'utilisateur
  • Création des comptes courant et épargne avec un taux
  • Faire des crédit, débit et virement sur les différents comptes bancaires
  • Saisir le Découvert bancaire autorisé lors de la création d'un compte courant

Technologies utilisées :

  • Java
  • MySql
  • Hibernate

Pattern:

  • Mvc

IDE:

  • Netbeans

Serveur:

  • Xampp

Deadline do projet : une semaine(Lundi-Vendredi).


Lors d’avoir reçu le sujet du projet, j’étais censé à développer la base de données et la valider avec le professeur.

Une fois cette étape conclue, j’ai commencé le développement direct avec NetBeans.

La première chose que j'ai développé c'était des interfaces avec la bibliothèque Swing sur le mode design de NetBeans.

De suite j'ai développé la fonctionnalité "création des utilisateurs" et la possibilité de les modifier ainsi comme la création et modifications des comptes.

Pour finaliser j'ai développé la possibilité des réaliser des crédits, débits et virements entre les comptes.


Mon point de Vue

Ce projet a servi pour consolider mes connaissances sur la programmation orienté objet que je venais d'apprendre sur le précèdent projet Gestion d'élèves.
Pour la première fois, j'ai utilisé un framework (Hibernate) pour effectuer la communication avec la base de données et faire générer mes classes.
Aussi pour la première fois, j'ai eu le plaisir de programmer en Java et reconnaitre ça puissance et avantages au tant que langage orienté objet comme le fait de pouvoir développer pour différentes plateformes.
